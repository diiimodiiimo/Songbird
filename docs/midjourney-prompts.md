## Prompt Variations & Tips

### For Active States (Icons)
Add to base prompt:
```
, slightly thicker stroke, subtle glow effect
```

### For Different Sizes
Change aspect ratio:
- Icons: `--ar 1:1`
- Empty states: `--ar 9:16` or `--ar 4:3`
- Wide formats: `--ar 16:9`

### For Consistency
Reference existing logo:
```
, matching SongBirdlogo.png style and texture
```

### For Refinement
If initial result needs adjustment:
- Add `, more minimal` for simpler designs
- Add `, warmer tones` for color adjustment
- Add `, darker background` for contrast
- Add `, more texture` for grain emphasis

---

## Batch Generation Strategy

1. **Generate base set** with all prompts
2. **Review for consistency** across mood birds
3. **Refine individual assets** that don't match
4. **Create active states** by modifying base icons
5. **Export at all sizes** (base, @2x, @3x)

## Quality Checklist

Before finalizing assets:
- [ ] Transparent background
- [ ] Fine grain texture visible
- [ ] Dark mode optimized (light on dark)
- [ ] No emoji elements
- [ ] Emotion felt, not cartooned
- [ ] Consistent with existing logo
- [ ] Warm but muted tones
- [ ] Premium but warm feel
