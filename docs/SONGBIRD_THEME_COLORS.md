## Midjourney Prompt Template

Use this template for generating SongBird logos in different theme colors:

```
A minimalist songbird logo, [BIRD_NAME], featuring [PRIMARY_COLOR] and [ACCENT_COLOR] 
on a [BACKGROUND_COLOR] background, modern app icon style, clean vector design, 
simple silhouette, elegant and sophisticated --ar 1:1 --v 6
```

### Example Prompts:

**Northern Cardinal:**
```
A minimalist songbird logo, Northern Cardinal, featuring vivid crimson #C41E3A 
and soft coral pink #FF6B6B on a near black #141414 background, modern app icon style, 
clean vector design, simple silhouette, elegant and sophisticated --ar 1:1 --v 6
```

**Eastern Bluebird:**
```
A minimalist songbird logo, Eastern Bluebird, featuring soft sky blue #6CA0DC 
and warm peach #E8A87C on a deep navy #0F1520 background, modern app icon style, 
clean vector design, simple silhouette, elegant and sophisticated --ar 1:1 --v 6
```

**Painted Bunting (Premium):**
```
A minimalist songbird logo, Painted Bunting, featuring a gradient of blue #0066CC 
to green #00AA44 to red #CC0033 on a deep teal #0A1214 background, 
modern app icon style, clean vector design, multicolor bird silhouette, 
elegant and sophisticated, premium look --ar 1:1 --v 6
```
